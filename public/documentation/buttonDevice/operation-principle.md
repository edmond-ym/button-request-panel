# Operation Principle

When a button is prssed, the API request is sent to and processed by our system.

## Sending API Request

|                        |                                                                    |
| ---------------------- | ------------------------------------------------------------------ |
| Link                   | {{ baseUrl }}/deviceAPI/v1/{Device Id}?button_id={Button Id}       |
| Authorization Method   | Bearer Token                                                       |
| Request Method         | POST                                                               |
